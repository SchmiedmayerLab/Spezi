# Spezi Scheduler


The Scheduler module allows developers to schedule tasks based on predefined schedules.

For more information, please refer to the [API documentation](SpeziScheduler.docc/SpeziScheduler.md).


## The Spezi Template Application

The Spezi Template Application provides a great starting point and example using the Spezi Scheduler module.
